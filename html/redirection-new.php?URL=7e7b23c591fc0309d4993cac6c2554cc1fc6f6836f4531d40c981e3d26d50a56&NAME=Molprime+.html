    Description

    MolPrime+ is a chemical structure drawing tool based on the unique sketcher from the Mobile Molecular DataSheet (MMDS).

    Features:

    - draw chemical structures

    - send structure data via email

    - open structures from email or web

    - create graphical images or Microsoft Word documents with embedded structure graphics

    - calculate properties based on structures

    - search public databases (ChEBI, PubChem) by structure or name
